How To Use It

  1. Start with the Opening part.
  2. Play one line and repeat it aloud.
  3. When ready, use Play Part to hear the whole section in order.
  4. Use Time Practice last for difficult clock expressions.

Pronunciation Rules

  • CMEL: say “C, M, E, L” first, then “see-mel”.
  • LKS: say “L, K, S”.
  • HKU: say “H, K, U”.
  • 12:25 p.m.: say “twelve twenty-five p.m.”
